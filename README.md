# BasketballServiceSystem
# 项目介绍
项目背景是基于NBA开发的智能篮球服务系统，使用了Vue+Springboot+deepseek。项目功能分为两部分：用户和管理员，其中用户可以查询球赛信息，查询球员信息，查询球队信息，查看赛事直播信息，智能分析用户在球场适合的位置，智能问答。管理员的可以对赛事，球队，球员等信息进行管理
Java项目，课程设计，毕业设计源码
